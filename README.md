# 🎮 GameKU

> Platform marketplace digital modern untuk jual beli game, top up, Steam Key, voucher, akun game, dan layanan digital lainnya yang dilengkapi dashboard reseller, sistem verifikasi, autentikasi Google, pembayaran online, serta **StreamKU** sebagai pusat streaming anime, donghua, drakor, dan drama China.

# 🚀 Instalasi

Clone repository

```bash
git clone https://github.com/maizarsec1337/GameKU.git
```

Masuk ke folder project

```bash
cd GameKU
```

Jalankan installer otomatis

```bash
python3 jalan.py
```

Installer akan secara otomatis:

- Mendeteksi sistem operasi
- Mendeteksi nama user desktop
- Memeriksa Python
- Memeriksa Node.js
- Memeriksa npm
- Memeriksa Git
- Memasang seluruh dependency
- Membuat folder storage yang diperlukan
- Membersihkan cache yang aman
- Menjalankan aplikasi

---

# ▶️ Menjalankan Project

Apabila installer telah selesai dijalankan:

```bash
npm run dev
```

Frontend
